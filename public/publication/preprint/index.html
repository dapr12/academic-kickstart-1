Scalability and cost-effectiveness analysis of whole genome wide association studies on Google Cloud Platform and Amazon Web Services

Ines Krissaane, Carlos De Niz, Laura Versmee, Alba Gutiérrez-Sacristán, Gabor Korodi, Nneka Ede, Ranjay Kumar, Hermine Hovhannisyan, Stephen Fang, Nick Ragusa, Chris Noonan, Jessica Lyons, Arjun Manrai, Chirag Patel, Isaac Kohane, Paul Avillach
April 2019
PDF Code Dataset Custom Link

Abstract

We provide a straightforward and innovative methodology to optimize cloud configuration to conduct genome wide association studies with Spark clusters on both Google Cloud Platform and Amazon Web Services with Hail genomic variant store. By deploying and benchmarking numerous different cluster configurations in the cloud, we present essential breakthroughs by pointing out the best compromise between speed and cost for performing a genome wide association study on four distinct datasets.
